        <h2>⚙️ Como Configurar</h2>
        <ol>
            <li>Clone o repositório: <code>git clone https://github.com/username/forumhub_api.git</code></li>
            <li>Instale as dependências utilizando Maven.</li>
            <li>Configure o banco de dados no arquivo <code>application.properties</code>.</li>
            <li>Inicie o servidor: <code>mvn spring-boot:run</code>.</li>
